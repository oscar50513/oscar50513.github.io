撰寫GUI設計流程

  1. 建立主視窗（大小、位置、名稱）
  2. 元件（按鈕、文字方塊、選單）加入視窗中
  3. 事件處理函示（event handler)：使用者互動時的觸發行為

BMI App

  1. 提供輸入框讓使用者輸入身高(m)體重(kg)
  2. 點擊計算按鈕後計算ＢＭＩ指數
  3. 顯示結果於畫面上

建立 BMI App 主視窗

 1. 建立視窗: 
     - tk.Tk()
 2. 設定標題、大小、背景:
     - tk.title("")
     - tk.geometry('')
     - tk.configure(background = '')

建立 BMI App 元件

1. 標題顯示ＢＭＩ計算器（文字元件）
2. 身高體重輸入區塊
3. 顯示結果按點擊按鈕

建立 BMI App 事件處理

1. 定義功能(建立函式)： 
    - def calculate_bmi_value()
    - def bmi_status_description()
2. 將函式綁定至元件按鈕上： command
