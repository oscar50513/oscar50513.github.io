進階電腦操作方法_CMD

  1. 關於 Windows 檔案系統

    • 是一個 階層式 的結構
    • 檔案總管(File Explorer)
      • 點選 本機 可以看到目錄和磁碟機
    • 磁碟機 / 根目錄 (Disk/Root Folder)
      • 用 磁碟機代號 來區隔電腦中的儲存空間
      • C: / D: / E: / F: /
    • 目錄 (Folder) 與檔案 (File)
      • 資料夾 <-> 目錄
      • 路徑( / )
    • 可執行檔(Executable File)
      • .exe

  2. 關於 Windows 命令提示字元(cmd)

    • 圖形介面 與 命令介面
      • 圖形介面 : 讓使用者方便使用，會隱藏一些細節
      • 命令介面 : 更接近電腦運作的本質
    • 進入 命令提示字元
      • 在搜尋欄輸入 cmd
      • 在檔案路徑欄上輸入 cmd
      • 當前工作路徑:
    • 切換磁碟機
      • 輸入 磁碟機代號 加上 冒號:
    • 切換工作目錄
      • 輸入 cd \ : 回到根目錄
      • 輸入 cd 目錄名稱: 進入該目錄資料夾
    • 查詢工作目錄
      • 輸入 cd dir : 查詢當前所有目錄及檔案
    • 開啟檔案 / 啟動可執行檔
      • 在適當的路徑下打檔案名稱，便可以啟動應用程式
